#  Notes App (Full Stack Project)

A simple full stack Notes Application built using **Node.js, Express, and EJS**.  
This app allows users to create, read, update, and delete notes (CRUD operations).

---

##  Features

-  View all notes
-  Create new note
-  View single note
-  Edit note
-  Delete note

---

##  Tech Stack

- Backend: Node.js, Express.js
- Frontend: EJS (Embedded JavaScript Templates)
- Middleware:
  - method-override
  - uuid
- Styling: CSS (public folder)
